# Demo of TypeScript declaration map used with SCSS

This would be a great addition to the [`dts-css-modules-loader`](https://github.com/Megaputer/dts-css-modules-loader).

For more details see:  
https://github.com/Megaputer/dts-css-modules-loader/issues/27

## Go To Definition

In VSCode:

1. Open the index.ts file.
2. Right click on either "wrapper", "foo" or "bar" and then choose "Go To Definition (F12)"
